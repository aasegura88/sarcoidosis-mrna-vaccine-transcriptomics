# Sarcoidosis mRNA Vaccine Transcriptomics

Code and figure-generation workflows for the manuscript:
**“mRNA vaccination reveals a stable interferon-biased immune state that constrains transcriptional responses in sarcoidosis.”**

## Repo structure
- `scripts/` method-centric analysis code
- `notebooks/` figure-centric notebooks
- `figures/` rendered manuscript figures
- `results/` intermediate outputs (no sensitive data)
- `data/` templates and documentation (no raw/subject-level data)

## Data availability
Raw sequencing data and subject-level metadata are not included in this public repository.
Processed inputs required to reproduce analyses are available from the corresponding author upon reasonable request.
